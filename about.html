Welcome To Aesthetic Lab!


Aesthetic lab is a website dedicated for different artists to be featured in and gain recognition. The website focuses solely on the art content only. Different types of art can be featured on the website such as photography, digital art, illustrations, designs, short movies etc.


This is website is a university project for web designing course at Near East University of Northern Cyprus. No monetization system is officially implemented. The e-commerce feature in the submission is not fully working as it's just a requirement for the project for display purposes.
